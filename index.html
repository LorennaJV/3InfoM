Versionamento

o git é um software que permite realizar versoes de um projeto. o github desktop é um software que faz a ligação entre visual studio code, git e github.

o github desktop permite fazer a sincronia entre o repositorio local e o repositorio remoto.

commit

é um processo de confirmação de uma versão. ou seja, a cada alteração é possivel confirmar a alteração

push

esse comando envia (sincroniza) as alteraçãoes realizadas no repositorio local com o repositorio remoto no github.
